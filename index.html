Production VPS Infrastructure

Docker Traefik Cloudflare Ubuntu

Problem

Needed a reliable, secure platform to host multiple web applications and services without paying for expensive managed solutions.

Architecture

Single VPS running Docker containers behind Traefik reverse proxy. Cloudflare provides DNS, DDoS protection, and SSL termination. All services defined in Docker Compose for reproducible deployments.

What I Built

  • Configured Traefik with automatic SSL certificate management via Let's Encrypt
  • Set up Cloudflare Tunnels for secure ingress without exposing ports
  • Implemented container-level network isolation
  • Created automated backup scripts for critical data

Security Considerations

Fail2ban for SSH protection, UFW firewall rules, non-root container execution, secrets management via environment files excluded from version control.

n8n Automation Platform

n8n Docker REST APIs Webhooks

Problem

Manual, repetitive tasks across multiple platforms were consuming hours each week. Needed a self-hosted automation solution with full data control.

Architecture

Self-hosted n8n instance running in Docker, connected to external services via API integrations. Webhook endpoints for real-time triggers, scheduled workflows for batch operations.

What I Built

  • Automated data sync workflows between multiple platforms
  • Built notification pipelines for monitoring and alerting
  • Created API integrations with third-party services
  • Implemented error handling and retry logic for reliability

Scalability

Workflows designed for idempotency to handle retries safely. Queue-based execution prevents overwhelming external APIs. Modular workflow design enables reuse across different automations.
