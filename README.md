# Jaedon Chin - Portfolio

Personal portfolio website showcasing cybersecurity, automation, and systems engineering work.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, modern layout (Grid/Flexbox)
- **JavaScript** - Minimal, progressive enhancement only

## Project Structure

```
├── index.html          # Main portfolio page
├── css/
│   └── style.css       # All styles (variables, reset, components)
├── js/
│   └── main.js         # Minimal interactivity
└── assets/
    ├── images/         # Project screenshots, profile photo
    └── icons/          # SVG icons
```

## Development

No build step required. Open `index.html` in a browser or use a local server:

```bash
# Python 3
python -m http.server 8000

# Node.js (if npx available)
npx serve
```

## Deployment

Static files - deploy to any web server, CDN, or VPS.
